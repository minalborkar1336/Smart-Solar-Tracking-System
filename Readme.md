\# 🌞 Smart Solar Tracking System with Rain Sensor and IoT Monitoring



\## 📌 About The Project



This project is an IoT-enabled Smart Solar Tracking System designed to improve solar panel efficiency by automatically adjusting the panel orientation according to sunlight intensity.



The system also includes rain detection and an automatic cleaning mechanism to maintain the performance of the solar panel.



Real-time monitoring of solar voltage, sunlight intensity, and rain conditions is achieved using the ThingSpeak IoT cloud platform.



\---



\## ✨ Features



\- 🌞 Automatic solar tracking based on sunlight direction

\- 💧 Rain detection and automatic cleaning mechanism

\- 📡 IoT-based real-time monitoring

\- ⚡ Solar voltage measurement

\- 🔄 Servo-based panel movement

\- ☁️ Cloud data visualization using ThingSpeak

\## ⚙️ Working Principle



Two LDR sensors are used to detect the direction of maximum sunlight.



The Arduino Uno compares the LDR values and rotates the servo motor towards the brighter side, allowing the solar panel to receive maximum sunlight.



The rain sensor detects rainfall conditions and activates the wiper mechanism to clean the panel surface.



The NodeMCU ESP8266 sends sensor readings to ThingSpeak for remote monitoring and analysis.
